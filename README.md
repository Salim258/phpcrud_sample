#CRUDAPP_sample

Introduction - This is an Sample basic PHP Application where User can perform CRUD Operations

How to use it ?

1 - Creat a database in phpmyadmin

2 - Then import crud.sql file into created database 

3 - Then edit in the process.php use your username, password, and database name that you've created earlier.

4 - Finally start creating your records in db using the application or manually in the DB. Once populated with record then you can edit, delete, or add new records using application.

This is an opensourced application can be effectively used for learning and demonstration purposes.
